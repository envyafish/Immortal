# Changelog

本文件记录 Immortal 面向用户的版本更新说明。

## 1.3.2 🎨

- 🎨 调整前端部分组件样式

## 1.3.1 🔧

- 🛠️ 修复若干已知问题，提升订阅与版本检测相关流程的稳定性
- 🤖 优化 TG / Discord 交互判断，番号消息会优先按订阅流程直接处理
- ✨ 调整 `AvBase 上新同步`，改为以当天为基准，按前后偏移天数同步范围内的影片
- ⚡ 新增影片订阅后立即异步执行一次订阅下载
- 🎛️ 调整榜单过滤方式，将站点切换抽离到页面顶部
- 📝 已保存的 `AvBase 上新同步` 任务需要重新配置参数，原 `offset_days` / `target_date` 不再生效

## 1.3.0 🚀

- ✨ 新增版本检测推送功能
- 🛠️ 增强获取本地版本号
- 📝 版本检测推送功能需要在定时任务中手动配置后才会生效

## 1.2.1 🔧

- 🛠️ 修复多个入口订阅影片提示缺失索引的异常

## 1.2.0 🚀

- 🛠️ 修复清理小文件工具的参数异常
- ✨ 新增版本更新检查功能
- 📝 新增更新日志记录

## 1.1.8 🔧

- 🛠️ 修复 Transmission 下载错误异常
